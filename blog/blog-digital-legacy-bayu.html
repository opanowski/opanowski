Digital Legacy: Jurus Amankan Kenangan Si Bontot Biar Abadi di Dunia Maya

8 Mei 2026 · Om Opan — Ciracas, Jakarta Timur · 📓 LOG #007

Pernah kepikiran gak sih, kalau kita kehilangan orang tersayang — yang tersisa selain doa itu apa? Yup, file digital. Foto pas dia lagi konyol, video ketawa bareng, sampe chat-chat gak jelas di WhatsApp. LOG #007 ini mungkin yang paling personal buat gw. Misi "007: Skyfall-nya Bunker Opanowski." 1. Kumpulin Semua "Serpihan" Kenangan

Jangan biarkan foto-foto doi mencar-mencar. Ada yang di HP emak, ada yang di laptop lama, ada yang di kartu memori yang udah berdebu. Tarik semua ke satu folder master. Kasih nama yang jelas, misal Bontot_Archive. Jangan cuma 'New Folder (1)', ntar lo bingung sendiri pas nyari!

"Ayo kumpulin semua! Dari HP emak sampe kartu SD berdebu — gak ada yang boleh ketinggalan! 💪"

📸 Si Bontot Bayu. Kenangan yang selalu bikin senyum.
📸 Ekspresi ceria Si Bontot pas lagi ketawa lepas. Ini kepingan puzzle yang bikin log ini akhirnya bisa rilis.

2. Backup Pake Rumus 3-2-1 (Biar Gak Nangis Darah!)

Ini rumus wajib buat para tech-user kayak kita. Jangan cuma simpan di satu tempat:

📦 Rumus Backup 3-2-1
3️⃣ 3 Salinan — Bikin 3 copy data utama. Minimal, bukan opsional.
2️⃣ 2 Media Beda — Simpan di media fisik yang beda. Misal di Harddisk Eksternal satu, di Cloud (Google Photos/Drive) satu.
1️⃣ 1 Lokasi Terpisah — Taruh satu copy di luar rumah (alias di Cloud tadi). Jadi kalau (amit-amit) laptop lo error, kenangan Si Bontot tetep aman di "langit".

3. Akun Sosmed? Mode Kenangan On!

Sosmed almarhum itu gudang memori. Biar gak ilang atau disalahgunakan orang iseng, atur Legacy Contact di Facebook buat ubah akun jadi 'Memorialized Account'. Jadi orang masih bisa liat foto-fotonya buat lepas kangen, tapi akunnya gak bakal bisa diacak-acak orang asing.

"Legacy Contact? Kontak warisan? Fitur apaan ini? — Eh tunggu, ini penting banget ternyata. 🤔"

🔒 Cara Set Legacy Contact Facebook
1. Buka Pengaturan & Privasi → Pengaturan
2. Scroll ke Informasi Umum → Kelola Akun
3. Pilih Kontak Warisan → tambahkan nama terpercaya
4. Atau minta keluarga lapor ke Facebook buat mode Memorialized
📸 Momen hangat almarhum Bayu bareng Yani, adik gw nomor 3. Kenangan yang gak ternilai harganya.

4. Bikin "Bunker" Khusus di GitHub

Karena gw lagi hobi mainan GitHub Pages, gw buatin satu halaman khusus buat Bayu. Isinya galeri foto pilihan sama cerita singkat. Kelebihannya?

✅ Kenapa GitHub Pages?
✅ Gratis — nol biaya hosting tahunan
✅ Ringan & cepat — static site, no database
✅ Awet — selama GitHub ada, halaman itu live
✅ Custom — bisa tampilin galeri, cerita, video embed
✅ Version control — gak ada yang bisa terhapus sembarangan

Selama GitHub masih ada, kenangan itu bakal tetep live tanpa perlu bayar hosting mahal-mahal tiap tahun, bray!

📌 Kesimpulan

"Kenangan di hati, data di cloud, halaman di GitHub. Setup-nya beres. Lo tetep ada, Yu. 😎"

Mengamankan data Si Bontot itu cara gw buat bilang: "Lo tetep ada, Yu." Kenangan emang di hati, tapi file digital itu bukti sejarah yang harus kita jaga dengan presisi.

Empat langkah tadi — kumpulin serpihan, backup 3-2-1, legacy contact sosmed, sama bunker GitHub — itu bukan overkill. Itu standar minimum buat generasi kita yang hidupnya udah separuh di dunia digital.
